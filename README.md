---

## 📋 기획 개요

서울의 다양한 음식 레시피를 카테고리, 맛 프로필, 매운맛 단계별로 필터링할 수 있는 웹 앱. 한국 음식에 익숙하지 않은 외국인을 포함한 다양한 사용자를 위해 EN/KR 언어 전환을 지원합니다.

---

## ✨ 주요 기능

- **카테고리 필터** — 음식 종류별 분류 탐색
- **맛 프로필 필터** — 달콤함, 짭짤함, 매콤함 등 복합 맛 태그 선택
- **매운맛 단계 조절** — 1~5단계 슬라이더로 원하는 매운맛 범위 설정
- **EN / KR 전환** — 상단 토글 버튼으로 언어 즉시 전환
- **필터 초기화** — 한 번에 모든 필터 리셋

---

## 🛠️ 기술 스택

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
